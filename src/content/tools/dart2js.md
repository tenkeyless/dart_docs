# title: "dart2js: Dart-to-JavaScript compiler"
title: "dart2js: Dart-to-JavaScript 컴파일러"
short-title: dart2js
# description: The dart2js compiler compiles Dart code to deployable JavaScript.
description: dart2js 컴파일러는 Dart 코드를 배포 가능한 JavaScript로 컴파일합니다.
---

:::version-note
Dart 2.18은 Dart 패키지에서 `dart2js` 명령줄 도구를 제거하지만, 
dart2js 컴파일러는 유지합니다.

[`dart compile js`](/tools/dart-compile#js)를 사용하여, 
Dart 코드를 배포 가능한 JavaScript로 컴파일합니다.
:::
