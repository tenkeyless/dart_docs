모든 플랫폼에서 고품질 앱을 위한 접근성 있고 휴대성이 뛰어나며 생산적인 언어

Dart는 무료이며 오픈 소스입니다  

쉬운 접근성
null 세이프티 및 패턴 매칭과 같은 최신 기능을 제공하는, 일관되고 간결하며 강력하게 타입화된 프로그래밍 언어로 개발하세요.
생산적인 개발
변경 사항을 반복적으로 적용하세요. 핫 리로드를 사용하면 실행 중인 앱에서 즉시 결과를 확인할 수 있습니다.
모든 플랫폼에서
휴대성이 뛰어나고 빠름
모바일, 데스크톱 및 백엔드용 ARM, x64 및 RISC-V 머신 코드로 컴파일합니다. 또는 웹용 JavaScript 및 WebAssembly로 컴파일합니다.
쉬운 접근성
  • 이벤트 기반(event-driven) 코드를 포함하는 사용자 인터페이스에 대한 성숙하고 완전한 async-await는 isolate 기반 동시성과 결합됩니다.
  • 사운드 null 세이프티, 컬렉션 if 및 패턴 매칭과 같은 기능을 사용하여, 안전하고 간결한 코드를 작성합니다.
  • 배우기 쉽고 익숙한 구문을 갖춘 일관된 프로그래밍 언어입니다.
생산적인 개발
  • 핫 리로드를 사용하여 소스 코드를 반복적으로 변경하면, 실행 중인 앱에서 효과가 즉시 표시됩니다.
  • 풍부한 정적 분석(rich static analysis)과 강력하고 구성 가능한 도구를 갖춘 유연한 유형 시스템을 사용하여 코드를 작성하세요.
  • 원하는 코드 편집기로 프로파일링, 로깅, 디버깅 을 수행하세요.
모든 플랫폼에서 빠름
  • 즉시 시작(instant startup)할 수 있도록 앱을 네이티브 머신 코드로 AOT 컴파일합니다.
  • JavaScript와 WebAssembly를 위한 완전하고 성숙하며 빠른 컴파일러로 웹을 타겟팅하세요.
  • 단일 프로그래밍 언어로 작성된, 앱을 지원하는 백엔드 코드를 실행합니다.
